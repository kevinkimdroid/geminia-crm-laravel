What this document is

Plain summary of how the new Laravel CRM compares to the old Vtiger-based CRM: improvements, new features, and remaining gaps.

Project: Geminia CRM (Laravel)
Compared to: Legacy Geminia CRM (Vtiger: PHP, Smarty, vtiger MySQL database)

1. Executive summary

The new CRM replaces the older Vtiger-based stack with Laravel 11 and Blade, while keeping the same MySQL vtiger database for core CRM data. That avoids duplicate records and allows running both systems side by side during migration.

Where the new system is stronger

  • Modern framework, easier long-term maintenance
  • Stronger web security (e.g. CSRF, structured auth, password reset)
  • Documented deployment (Composer, build step, migrations)
  • Business-specific features: ERP clients, maturities, marketing, PBX, complaints, ticket SLA tooling

Where legacy may still cover more

  • Full calendar parity, products/quotes/orders/invoices in Laravel
  • Deep Vtiger items unless rebuilt: workflows, custom views, related lists, full document/email parity

3. Core CRM in Laravel today

  • Authentication — login, logout, password forgot/reset
  • Dashboard — Vtiger data where applicable
  • Contacts — CRUD, follow-ups, campaigns
  • Leads — full CRUD
  • Tickets — CRUD, export, close, reassign, comments
  • Deals — full CRUD
  • Activities — create/list + APIs
  • Search — global search
  • Support / customers — ERP-aware views, serve-client flows
  • Reports — SLA, aging, tickets by date, exports, etc.
  • Settings (partial) — users, departments, groups, automation, PBX, SLA, layout, roles

MIGRATION_PLAN.md in the repo is an older checklist; some “pending” rows there are already implemented.

5. Gaps to plan for

Legacy Vtiger may still be needed until Laravel covers:

  • Full calendar/events UI parity
  • Products, quotes, sales orders, invoices
  • Deep parity: quick create everywhere, custom views, related lists, import/export everywhere, documents, full email client, workflows, cron, dashboard widgets like Vtiger

6. One-line conclusion

The Laravel CRM modernizes the platform, adds insurance and operations features, and shares one database with Vtiger; full replacement of every legacy module remains a phased effort.
